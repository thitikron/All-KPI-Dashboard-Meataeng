Google Sheets Sync

⚫Local Only
📋 วิธีตั้งค่า (ทำครั้งเดียว — ไม่ต้องแก้โค้ด)
  1. เปิด script.google.com → New Project
  2. ลบโค้ดเดิมออก → วางโค้ดจากไฟล์ Code.gs ที่ดาวน์โหลด
  3. กด Deploy → New Deployment → Web App
    Execute as: Me  |  Who has access: Anyone
  4. Copy Web App URL มาวางในช่องด้านบน → กด "บันทึกการตั้งค่า"
  5. กด "🔌 ทดสอบการเชื่อมต่อ" → ต้องขึ้น 🟢 Synced
  6. กด "🔧 Setup Google Sheets (ครั้งแรก)" → ระบบสร้าง Sheet ใน Google Drive อัตโนมัติ พร้อม Link เปิดดูได้เลย
✅ ไม่ต้องสร้าง Google Sheet เอง และไม่ต้องแก้โค้ดใดๆ ทั้งสิ้น
